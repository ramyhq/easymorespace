## What It Does

Compress your images intelligently — resize, rename, preserve everything. No ads. No tracking. No paywalls.

- Input: **JPG, PNG, WebP, BMP, TIFF**
- Output: **JPG, PNG, WebP, AVIF**
- EXIF metadata fully preserved (camera, date, GPS)
- Zero rotation — orientation stays intact
- Single-file .exe / .app — **no installation required** (~18 MB)

## Key Features

| Feature | Detail |
|---------|--------|
| Smart Compression | Adjustable quality slider, resize to max dimension |
| Two Modes | Folder Mode (entire folder) or Files Mode (hand-picked) |
| Batch Rename | Pattern tokens `{prefix}_{date}_{num}` with smart date extraction & live preview |
| Two Workflows | Compress + Rename, or Rename Only (no compression) |
| Activity Log | Real-time per-file log — green/red/yellow coloring |
| Languages | English, Arabic, French |
| Offline | Works fully offline — zero data leaves your device |
| Ad System | Rotating image-only banner from `adsimages/ad.json` |

## Troubleshooting

### Windows
| Problem | Solution |
|---------|----------|
| "Windows protected your PC" | Click **More info** → **Run anyway** |
| "VCRUNTIME140.dll not found" | Install [VC++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe) |
| "Another instance is running" | Task Manager → End `EasyMoreSpace` task |

### macOS
| Problem | Solution |
|---------|----------|
| "App is damaged / can't be opened" | Run: `xattr -cr /path/to/EasyMoreSpace.app` |
| App not in Launchpad | Drag `.app` to `/Applications` folder |
| "Another instance is running" | Run: `pkill -f EasyMoreSpace` |

Full troubleshooting: [TROUBLESHOOTING.md](TROUBLESHOOTING.md)
